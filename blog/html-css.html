The difference between Margins, Borders and Padding

The most common computer programming language that is used for the development and creation of websites and web applications is, HTML. This stands for Hyper Text Markup Language.

Websites display and are created using a combination of elements i.e headings, images, text/paragraphs etc

These elements are identified using HTML opening and closing tags e.g. (looking below) the paragraph tags with the paragraph text (content) inside the tags.

The following shows an image of a websites HTML code and the corresponding browser output displayed:

The text in orange are the HTML tags that identify the element. Within those tags you can add whatever content. Some tags/elements require specific content of that type i.e. an img tag/element will only output images.

Margins Borders and Padding are examples of elements. looking at the image below, Marigns, borders and padding refer to the spacing around elements/content.

We can see the 'space' around elements is divided into three groups/layers. The inital layer/space around the content is the Padding. The next layer, that refers to the space around the padding, is the Border. Followed by outer most layer of space around an element which is the Margin.

Margins, Borders and Paddings can be manipulated and changed to whatever liking, using CSS.
