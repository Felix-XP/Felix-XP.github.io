The difference between Margins, Borders and Padding

  • Websites and web applications are created using a combination of HTML, CSS and JS codes(languages).
  • You can think of HTML as the strutural foundation of websites, responsible for holding everything together.
  • CSS and JS are responsible for the websites aestitics and dynamics.

When displayed is a code editior, we can see that, websites contain and are functional using a range of different elements.

These elements are identified using HTML opening and closing tags e.g. (looking below), the 'p' tags indicates to the browser that the follwing content is a 'paragraph'

The following shows an image of a websites HTML code, in a code editor, and the corresponding browser output displayed:

The text in orange are the HTML tags that identify the element. Within those tags you can add whatever content. Some tags/elements require specific content of that type i.e. an img tag/element will only output images.

Furthermore, elements can be given characteristics by manipulating the elements attributes.

Margins Borders and Padding are examples of an elements attributes. looking at the image below, Marigns, borders and padding refer to the spacing around elements/content.

We can see the 'space' around elements is divided into three groups/layers. The inital layer/space around the content is the Padding. The next layer, that refers to the space around the padding, is the Border. Followed by outer most layer of space around an element which is the Margin.

Margins, Borders and Paddings can be manipulated and changed to whatever liking, using CSS and JS.
